## 每章 index.tex 详细说明

每个书目目录下都有 `index.tex`，每条记录格式为：
```latex
% [tikz|circuit] 来自 {源文件}:{行号}  {章节主题}
\input{附录_参考绘图/{路径}/{文件名}}
```

### 精确定位方法
1. 查本表找到章节 → 目录名
2. 读 `LaTeX_Project/附录_参考绘图/{书名}/{章节}/index.tex`（不存在时读上层 index.tex）
3. 按注释中的 `{源文件}:{行号}` 和 `{主题}` 定位具体图
4. 打开对应 `.tex` 文件阅读 TikZ/circuitikz 源码

### 使用场景

当需要绘制物理/数学图时：
1. 查阅上方表格找到对应章节
2. 打开 `index.tex` 找到具体图号和主题
3. 阅读对应 `.tex` 文件的 TikZ/circuitikz 源码
4. 参考 `em-drawing-patterns.md` 映射表翻译为 CeTZ
