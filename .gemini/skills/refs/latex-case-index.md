# LaTeX 自有绘图案例索引（315 tikzpicture）

本项目 `LaTeX_Project/` 中积累了 315 个经过验证的 TikZ 绘图案例，覆盖数学和物理两大学科。
遇到相似题型时，可参考对应 LaTeX 源码的绘图原理、坐标计算和样式技巧，翻译为 Typst/CeTZ。

## 路径规则
```
LaTeX_Project/{学科}/{大类}/{章节名}.tex
```

## 使用方式

遇到 Typst 新题时：
1. **按题型查表**：在上方找到最接近的分类
2. **打开 LaTeX 源码**：`LaTeX_Project/{路径}/{文件名}.tex`
3. **参考绘图原理**：坐标计算、元素布局、样式选择
4. **翻译为 CeTZ**：参考 `em-drawing-patterns.md` 的 TikZ→CeTZ 映射表
