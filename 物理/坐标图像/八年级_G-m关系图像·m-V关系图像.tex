\section{解答：探究重力大小与质量的关系图像}

\vspace{0.6cm}

\begin{center}
\begin{tikzpicture}[>=Stealth, scale=1.0]

  % 1. 绘制背景网格
  % 假设每个小格长宽为 0.4cm
  % 横向绘制 15 个小格，纵向绘制 12 个小格
  \draw[step=0.4cm, gray!60, line width=0.5pt] (0, 0) grid (6.0, 4.8);
  
  % 2. 绘制坐标轴
  % X 轴
  \draw[->, black, line width=1.0pt] (0, 0) -- (6.6, 0) node[below] {$m/(10^{-2}\,\text{kg})$};
  % Y 轴
  \draw[->, black, line width=1.0pt] (0, 0) -- (0, 5.4) node[left] {$G/\text{N}$};
  % 原点
  \node[below left] at (0, 0) {$0$};

  % 3. 绘制坐标轴刻度 (学生作图的得分点之一：合理标度)
  % X 轴：2 个小格 (0.8cm) 代表 5 个单位
  \foreach \x/\val in {0.8/5, 1.6/10, 2.4/15, 3.2/20, 4.0/25, 4.8/30} {
    \draw[black, line width=1.0pt] (\x, 0) -- (\x, -0.1);
    \node[below, font=\small, yshift=-0.05cm] at (\x, -0.1) {\val};
  }
  
  % Y 轴：2 个小格 (0.8cm) 代表 0.5 个单位
  \foreach \y/\val in {0.8/0.5, 1.6/1.0, 2.4/1.5, 3.2/2.0, 4.0/2.5, 4.8/3.0} {
    \draw[black, line width=1.0pt] (0, \y) -- (-0.1, \y);
    \node[left, font=\small, xshift=-0.05cm] at (-0.1, \y) {\val};
  }

  % 4. 描点
  % 根据表格数据：(5, 0.5), (10, 1.0), (15, 1.5), (20, 2.0), (25, 2.5)
  % 转换到画图坐标：(0.8, 0.8), (1.6, 1.6), (2.4, 2.4), (3.2, 3.2), (4.0, 4.0)
  \foreach \x/\y in {0.8/0.8, 1.6/1.6, 2.4/2.4, 3.2/3.2, 4.0/4.0} {
    \fill[black] (\x, \y) circle (2pt);
  }

  % 5. 连线
  % 用平滑直线穿过所有点，并稍微延伸原点出发
  \draw[black, line width=1.5pt] (0, 0) -- (4.8, 4.8);
  
  % 图号标注
  \node at (3.0, -1.2) {\Large 乙};

\end{tikzpicture}
\end{center}

\vspace{0.6cm}

{\small\color{gray}
\textbf{解题说明：}\\
1. \textbf{设置合理刻度：}在提供的空白网格上作图，得高分的第一步是为坐标系配上均匀且合理的标度。结合表二中 $G$ 和 $m$ 的最大数值，我们将横轴划分为“每双小格代表 $5\times 10^{-2}\,\text{kg}$”，纵轴划分为“每双小格代表 $0.5\,\text{N}$”。这样既充分利用了网格空间，点又刚好全部落在十字交点上，完美规避了估读描点造成的偏差。\\
2. \textbf{精确描点：}依照表格寻找对应的五组坐标点 $(5, 0.5)$、$(10, 1.0)$ $\cdots$ 到 $(25, 2.5)$，在图像上点上明显的实心小圆点。这是实验探究题作图的关键得分环节。\\
3. \textbf{归纳连线与结论补充解答：}由于这些点排列非常完美规律，因此使用直尺将这五点与原点 $(0,0)$ 连系成一条光顺的直线即可完成所有作图工作。这同时也直观有力地解答了第 (2) 问：从图像可以看出这是一条过原点的正比例函数线，由此得出结论：\textbf{物体所受重力的大小跟物体质量成正比}。
}

\vspace{0.6cm}

{\small\color{blue!70!black}
\textbf{绘图基本原理与步骤：}\\
\textbf{1. 精绘高质量网格底系：}TikZ 内置的高级 \texttt{\textbackslash draw[step=XX] grid} 功能极其适合用于此种带底纹坐标纸的物理试题渲染。在这里我不仅通过 \texttt{gray!60} 作为网格的填充色去高仿真模拟印刷书籍上的极其标志性的灰色效应底图，同时限定划定了 \texttt{15x12} 的精悍坐标系统网格宽幅，使排版极其轻量干净。\\
\textbf{2. 标度与数据的自动化张量推导：}如果逐个描刻全图标度坐标会导致代码呈爆炸式冗余且随时修改困难拉垮。为此我在坐标基轴的绘制中均埋入了自动化循环块 \texttt{\textbackslash foreach}。借用 \texttt{\textbackslash x/\textbackslash val in \{0.8/5, ...\}} 这种类字典形式对撞赋值输出直接完成一切“小刻度线加其对应文本”的批量渲染。更为彻底的是，甚至最后数据描点也全盘套用了一个 \texttt{foreach} 多元迭代数组，直接实现了完全的数学底层数据与展示视图解耦，让它呈现出了比肩一流 \texttt{python matplotlib} 的现代化图表出图全自动模块级优雅。
}

\section{附加图形：$\boldsymbol{m-V}$ (质量-体积) 图像基础网格}

\vspace{0.6cm}

\begin{center}
\begin{tikzpicture}[>=Stealth, line join=round, line cap=round, x=0.3cm, y=0.4cm]

  % 1. 绘制辅助虚拟网格 (细微等距虚线)
  % x 轴每 1 单位，y 轴每 1 单位
  \draw[cyan!60, dashed, line width=0.5pt, xstep=1, ystep=1] (0,0) grid (20,15);
  
  % 2. 绘制主网格 (粗实线)
  % 形成视觉正方形网格：xstep=4, ystep=3
  \draw[cyan!90!black, line width=0.8pt, xstep=4, ystep=3] (0,0) grid (20,15);
  
  % 3. 绘制带有箭头的黑框坐标轴 (横轴极度延展防拥挤)
  \draw[->, black, line width=1.0pt] (0,0) -- (0, 16.5) node[left, font=\large] {$m/\text{g}$};
  \draw[->, black, line width=1.0pt] (0,0) -- (24, 0) node[below, font=\large, xshift=0.3cm] {$V/\text{cm}^3$};
  
  % 4. 标注源点与各极坐标刻度数字
  \node[below left, font=\large] at (0, 0) {0};
  
  % X 轴主刻度
  \foreach \x in {4, 8, 12, 16, 20} {
    \node[below, font=\large, yshift=-0.1cm] at (\x, 0) {\x};
  }
  
  % Y 轴主刻度
  \foreach \y in {3, 6, 9, 12, 15} {
    \node[left, font=\large, xshift=-0.1cm] at (0, \y) {\y};
  }

  % 5. 增绘几种常见物质的 m-V 特征直线 (经过原点的正比例函数)
  % 铝：密度 2.7 g/cm³，m=15g 时，V=15/2.7 ≈ 5.56cm³
  \draw[red!80!black, line width=1.5pt] (0,0) -- (15/2.7, 15) node[above, font=\small\bfseries, yshift=2pt] {铝 $(2.7\,\text{g/cm}^3)$};
  
  % 水：密度 1.0 g/cm³，m=15g 时，V=15cm³
  \draw[blue!80!black, line width=1.5pt] (0,0) -- (15, 15) node[above right, font=\small\bfseries, xshift=2pt] {水 $(1.0\,\text{g/cm}^3)$};
  
  % 木块：密度 0.5 g/cm³，V=20cm³ 时，m=10g
  \draw[green!60!black, line width=1.5pt] (0,0) -- (20, 10) node[right, font=\small\bfseries, xshift=2pt] {木块 $(0.5\,\text{g/cm}^3)$};

\end{tikzpicture}
\end{center}

\vspace{0.6cm}
\hrule
\vspace{0.4cm}

{\small\color{gray}
\textbf{解题分析与说明：}\\
这是一幅典型的《质量与体积关系 ($m-V$) 图像》标准清空底面网格。纵坐标代表物体质量 $m$（单位：$\text{g}$），横坐标代表体积 $V$（单位：$\text{cm}^3$）。根据物理学中最基础的密度公式 $\rho = \frac{m}{V}$ 可知：若在其中描绘出经过原点的直线正比例函数图像，通过求解该代数直线解析式 $y = kx$ 获取的斜率参数 $k$，其本质所代表的物理属性正是该物质的密度 $\rho$。
}

\vspace{0.6cm}

{\small\color{blue!70!black}
\textbf{制图技巧与参数解释：}\\
\textbf{1. 非均匀坐标系等比化放缩：}仔细审视原版青色主网格：它在横向上跨越的标记单位量为 $4$，而在纵向爬升的单位量为 $3$。若单纯等比铺设（即令 $x$ 极距完全等于 $y$ 极距），这必然导致图中的网格变成扁平或高挑的长方形。为了挽救畸变、逼迫它在人眼视觉上重新聚合成横平竖直的正方形边框，我通过底层代码反向补偿设置了常数配置 $\mathtt{x=0.3cm, y=0.4cm}$。这波隐蔽的拉扯平衡操作，直接抹平了物理步长的差值： $4 \times 0.3\,\text{cm} = 1.2\,\text{cm}$ 以及 $3 \times 0.4\,\text{cm} = 1.2\,\text{cm}$，使其自动被封印压缩成了无比清爽治愈的正方形矩阵。\\
\textbf{2. 级联式双层网格宏算法（Dual Grid 叠加技巧）：}如果依靠纯手持写代码人工手绘这成百上千条纵横交错的格线，那势必将成为排版界的累赘。相反，本段代码直接祭出了 LaTeX 原生级别的极简阵列函数 \texttt{grid}！第一波用淡蓝虚线底漆 \texttt{cyan!60, dashed} 铺满以 $1$ 为极小单位间隔的全屏细网格底纹；紧跟其后，原地再次盖上一发深着色的实粗线 \texttt{cyan!90!black} 且挂载长跨度步长阵列 \texttt{xstep=4, ystep=3} ，强行把粗硬扎实的四方龙骨大骨架砸在图表上层框架！两句短小精悍的叠层绘制指令，轻松平推碾压出了极高规格尺寸的专业印刷防伪底稿观感！
}
