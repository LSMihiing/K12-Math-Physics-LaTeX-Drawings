# K12 教辅绘图项目——按绘图类型重构章节计划

## 背景

当前项目按 **学科 → 绘图大类 → 年级** 组织章节。每个年级文件内包含大量不同绘图子类型（如画角、画高、涂色等），导致同一类型的绘图分散在不同年级文件中，不利于检索。

本次重构目标：按 **学科 → 绘图大类 → 绘图子类型** 重新组织，使文档更像一本"K12数理习题答案绘制手册"，方便按绘图类型快速定位。
